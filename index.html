Before you install

System requirements

  • Windows 10 or Windows 11 (64-bit)
  • A microphone
  • An internet connection (required for the first launch; also used to check for updates)

How to install

  1. Step 1 — Download
    Click the download button above to get the .msi installer.
  2. Step 2 — SmartScreen warning
    Windows may show a message that says "Windows protected your PC" or "unrecognized app." This is expected. The installer is not yet code-signed, so Windows does not recognize it yet.

    Click More info, then click Run anyway.

    Note: This warning will appear again on every update. Each update is a new installer — there are no silent background patches.
  3. Step 3 — Install and launch
    Follow the on-screen steps to finish the installation, then open Global Pathways.
  4. Step 4 — First launch
    The first time you open the app, it will download a speech model (about 339 MB). This takes a few minutes and requires an internet connection. Later launches do not need to download it again.

Your privacy

Your practice audio is processed on-device — no audio recordings leave your computer. The app does connect to the internet to download the speech model on first launch and to check for updates.
